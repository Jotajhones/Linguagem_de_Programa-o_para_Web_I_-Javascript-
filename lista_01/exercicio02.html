Exercicio 02

Declaração de constantes (const)


const Pi = 3.14159;

const SemanasDoAno;

// A) Tentativa de reatribuir um valor

SemanasDoAno = [];

// B) Tentativa de declarar sem inicializar
const SemanasDoAno;

Por que o JavaScript irá falhar?

A) TypeError (Erro de Tipo): A tentativa de reatribuir um valor a SemanasDoAno resultará em um erro do tipo TypeError: Assignment to constant variable. Em JavaScript, a palavra-chave const cria uma referência de leitura apenas. Isso significa que, uma vez que a variável foi associada a um valor, esse identificador não pode ser sobrescrito com uma nova atribuição.

B) SyntaxError (Erro de Sintaxe): Na verdade, o código vai "quebrar" antes mesmo de chegar na linha da atribuição. A linha const SemanasDoAno; lançará imediatamente um SyntaxError: Missing initializer in const declaration. A linguagem exige estritamente que toda constante seja inicializada com um valor no exato momento de sua declaração, afinal, se não pudermos mudar o valor depois, criar uma constante vazia não faria sentido.
